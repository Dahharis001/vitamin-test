# 💊 Vitamin Deficiency Self-Test · Тест на дефицит витаминов

A lightweight, single-file web app that helps people identify a **possible**
vitamin or mineral deficiency based on the symptoms they actually feel.
Symptom-first flow (the user picks a problem, not a vitamin), with a
probability score and a recommendation to confirm via blood test.

> ⚠️ Educational tool — estimates probability, not a diagnosis. Not a substitute
> for a doctor. Created with input from a licensed pharmacist.

**Live demo:** https://dahharis001.github.io/vitamin-test/

---

## 🇷🇺 О проекте

Лёгкий веб-тест (один HTML-файл), который помогает определить **вероятный**
дефицит витаминов или минералов по симптомам, которые человек реально ощущает.

**Как работает:**
1. Человек выбирает свою проблему (сон, усталость, судороги, кожа и т.д.)
2. Отвечает на 3–4 уточняющих вопроса
3. Получает вероятность дефицита по каждому нутриенту + рекомендацию

Покрыто 14 проблем и 16 витаминов/минералов. Автор контента — провизор
с опытом работы в фармкомпаниях.

⚠️ Тест оценивает **вероятность**, а не ставит диагноз. Точный дефицит
подтверждается анализом крови. Не заменяет консультацию врача.

## 🛠 Tech

- Pure HTML / CSS / JavaScript — no frameworks, no build step
- Single self-contained file (`index.html`)
- Works offline, opens in any browser, mobile-friendly

## 🚀 Run locally

Just open `index.html` in a browser. That's it.
